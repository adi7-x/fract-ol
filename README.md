## 📌 Overview
A mathematical and graphical project that involves discovering complex numbers and the concept of optimization in computer graphics. Calculates and renders various fractals in real-time.

## 🚀 Features
- **Fractals Supported:** Mandelbrot, Julia, and Burning Ship (or others depending on the bonus).
- **Interactions:** Zoom in/out smoothly following the mouse position, pan with arrow keys.
- **Dynamic Julia:** Change Julia set parameters dynamically with mouse movement.
- **Color Cycling:** Smooth psychedelic color transitions.

## 🛠️ Usage
```bash
make
./fractol mandelbrot
./fractol julia <real> <imaginary>
```
